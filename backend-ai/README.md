# Backend AI — Exercício de uso de LLM

Projeto de exemplo para entender como podemos utilizar uma LLM em serviços de API. O objetivo dele foi simular um processo de chamada de API própria que irá pegar o prompt e repassar a uma LLM, que com o resultado irá devolver ao usuário. 


## Como rodar

1. Instalar dependências:
```sh
npm install
```
2. Executar:
```sh
npm start
```
3. Executar (DEV):
```sh
npm run start:dev
```
