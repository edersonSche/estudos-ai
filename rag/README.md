# RAG — Exercício de uso do RAG

Projeto de exemplo para compreender o uso de RAG. O objetivo dele é emular a criação de embeddings em um banco vetorial (jsonl -> memória) de pseudo-documentos e com isto verificar, através de uma pergunta, quais os são os principais conteúdos que mais se encaixam ao que foi solicitado.

## Como rodar

1. Instalar dependências:
```sh
npm install
```
2. Executar:
```sh
npm start
```
